# reimagined-dollop

**5분봉 OHLCV 시계열 데이터를 활용한 반등 예측 모델 구현 (LSTM 기반)**  
이 프로젝트는 초보자를 위한 시계열 딥러닝 학습 프로젝트입니다.

## 📌 프로젝트 목표
- OHLCV 데이터를 분석하여
- 향후 4시간 이내 1.5% 이상 반등할 가능성을 예측
- 딥러닝 모델(LSTM) 기반으로 시계열 예측 실습

## 🗂️ 디렉토리 구조
rebound-predictor-mini/
├── data/ # 샘플 OHLCV 데이터
├── preprocess/ # 라벨, 기술 지표 처리 코드
├── model/ # LSTM 모델 및 학습 코드
├── evaluation/ # 성능 평가 코드
└── README.md # 프로젝트 소개 파일

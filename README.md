# Rebound Predictor Mini

**📈 5분봉 OHLCV 시계열 데이터를 기반으로 향후 4시간 내 반등 가능성을 예측하는 딥러닝 프로젝트**

---

## 📌 프로젝트 개요

이 프로젝트는 시계열 데이터를 활용한 반등 예측 딥러닝 모델을 구현하는 것을 목표로 합니다.  
초보자가 이해할 수 있도록, 데이터 생성부터 전처리, 모델 학습, 평가까지 전 과정을 직접 구현하고 문서화합니다.

---

## 🎯 목표

- 5분봉 OHLCV 데이터를 생성하거나 불러오기
- 향후 4시간 내 1.5% 이상 상승이 있었는지를 기준으로 라벨 생성
- RSI, 이동평균선 등 기술 지표 계산
- 시계열 딥러닝 모델(LSTM)을 사용하여 예측
- 결과 평가 및 시각화

---

## 🗂️ 프로젝트 구조

```
rebound-predictor-mini/
├── data/                   # 샘플 OHLCV 데이터 (CSV)
├── preprocess/             # 기술 지표, 라벨 생성 코드
├── model/                  # 딥러닝 모델 구조 및 학습 코드
├── evaluation/             # 예측 결과 평가 및 시각화
├── main.py                 # 전체 실행 스크립트
└── README.md               # 프로젝트 설명 파일
```
